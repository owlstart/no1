css 테두리

css 테두리 속성

css border 속성을 사용하면 요소 테두리의 스타일, 너비 / 색상을 지정할수 있습니다.

CSS 테두리 스타일

border-style 속성은 표시 할 테두리 종류를 지정

    속성 값
  • dotted점선 테두리를 정의
  • dashed 점선 테두리를 정의
  • solid -단단한 테두리를 정의
  • double이중 테두리 정의
  • groove3D 그루브 테두리를 정의 합니다.
    효과는 테두리 색상 값에 따라 다릅니다.
  • ridge3D 능선 테두리를 저의
    효과는 테두리 색상 값에 따라 다릅니다
  • inset3D 삽입 테두리를 정의 효과는 테두리 색상 값에 따라 다릅니다
  • outset3D 시작 경계를 정의 하지 않습니다 효과는 테두리 색상 값에 따라 다릅니다
  • none 경계를 정의 하지 않습니다
  • hidden숨겨진 테두리를 정의
border-style 속성은1-4 개의 값을 가질수 있습니다.(상단 테두리/ 오른쪽 경계/아래쪽 경계 /왼쪽 경계/왼쪽 경계) https://www.w3schools.com/css/css_border.asp
