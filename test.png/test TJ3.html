<script>태그

HTML에서 자바 스크립트 코드 사이에 삽입해야 <script> 하고 </script> 태그 입니다

예

<script>
document.getElementById("demo").innerHTML="MY FIRST JsvaScript";</script>

JavaScript in Body

자바 스크립트 함수및 이벤트

JavaScript function 는 "호출"될때 실행될수 있는 JavaScript 코드 블록입니다.
예를 들어 사동자가 버튼을 클릭 할때 와 같이 이벤트가 발생하면 함수를 호출 할수 있습니다.

<head> 또는 <body>의 JavaScript

HTML문서에 원하는 수의 스크립트 배치를 할수 있습니다.
스크립트는 HTML페이지 <body>의 <head>섹션 또는 둘 다에 배치 할수 있습니다.

<head>의 JavaScript

JavaScriptfunction는 <head>HTML페이지 의 섹션에 배치됩니다.
버튼을클릭하면 함수가 호출됩닌다.

해보기

<!DOCTYPE html>
<html>
<head>
<script>
function myFunction() {
document.getElementById("demo").innerHTML = "Paragraph changed.";
}
</script>
</head>
<body>
<h1>A Web Page</h1>
<p id= "demo">A Paragraph</p>
<button type="button" onclick ="myFunction()" >Try it </button>
</body>
</html>
