CSS반응형 웹디자인-RWD

Responsive Web Design - Introduction

반응 형 웹 디자인은 모든 장치에서 웹 페이지를보기 좋게 만듭니다.

반응 형 웹 디자인은 HTML과 CSS 만 사용합니다.

반응 형 웹 디자인은 프로그램이나 JavaScript가 아닙니다.

웹 페이지는 데스크톱, 태블릿 및 전화 등 다양한 장치를 사용하여 볼 수 있습니다. 장치와 상관없이 웹 페이지가보기 좋고 사용하기 쉬워야합니다.
