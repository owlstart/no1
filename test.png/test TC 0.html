CSS 튜토리얼

CSS는 HTML 문서의 스타일을 설명하는 언어입니다.
CSS는 HTML 요소를 표시하는 방법을 설명합니다.
이 튜토리얼은 기본에서 고급까지 CSS를 알려줍니다.

CSS는 큰 문제를 해결

HTML은 결코 웹 페이지를 포맷하기위한 태그를 포함하지 않았습니다! HTML은 다음과 같이 웹 페이지 의 내용 을 설명하기 위해 만들어졌습니다 .

<h1> 이것은 제목입니다 </h1>

<p>이 단락입니다. </p>

<font>및 색상 속성과 같은 태그가 HTML 3.2 사양에 추가되면

웹 개발자에게 악몽이 시작되었습니다.

글꼴과 색상 정보가 모든 단일 페이지에 추가되는 대규모 웹 사이트의 개발은

길고 비용이 많이 드는 프로세스가되었습니다.
이 문제를 해결하기 위해 W3C (World Wide Web Consortium)는 CSS를 만들었습니다.
CSS가 HTML 페이지에서 스타일 서식을 제거했습니다!

CSS는 많은 작업을 저장합니다!

스타일 정의는 일반적으로 외부 .css 파일에 저장됩니다.
외부 스타일 시트 파일을 사용하면 하나의 파일 만 변경하여 전체 웹 사이트의 모양을 변경할 수 있습니다!

CSS 구문

선택기는 스타일을 지정할 HTML 요소를 가리 킵니다.

선언 블록에는 세미콜론으로 구분 된 하나 이상의 선언이 포함됩니다.

각 선언에는 CSS 속성 이름과 값이 콜론으로 구분되어 포함됩니다.

CSS 선언은 항상 세미콜론으로 끝나고 선언 블록은 중괄호로 묶습니다.
