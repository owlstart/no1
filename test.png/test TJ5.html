외부파일

스크립트는 외부 파일에 배치 할수도 있습니다.

외부 파일:myScript.js

function  myFunction() {
document.getElementById("demo").innerHTML = "pqrqgraph changed.";
}

외부 스크립트는 어러 웹 페이지에서 동일한 코드가 사용될대 실용적입니다.
JavaScript파일의 파일 확장자는 .js입니다
외부 스크립트를 사용하려면 스크립트 파일 이름을 태그 의 src소스 속성 에 넣으십시오. <script> .

예

<script src= "myScript.js" ></script>

<head>또는 원하는 <body>대로 외부 스크립트 참조를 배치 할수 있습니다.
스크립트는 <script>태그가 있는 위치에 있는 것처럼 동작 합니다.

외부 스크립트는 <script> 태그를 포함 할수 없습니다

외부 JavaScript 장점

외부 파일에 스크립트를 배치하면 몇 가지 장점이 있습니다.

  • HTML 과 코드를 분리 합니다
  • HTML 및 JavaScript를 보다 쉽게 읽고 유지 관리 할수 있습니다.
  • 캐시 된 JavaScript 파일은 페이지오드 속도를 높일수 있습니다.

외부 참조

외부 스크립트는 전체 URLstrong> 또는 현재 웹 페이지에 상대적인 경로를 사용하여 참조 할 수 있습니다. 이 예제는 전체 URL을 사용하여 스크립트에 연결합니다.

한 페이지에 여러 스크립트 파일을 추가하려면
여러 스크립트 태그를 사용하십시오.

예
