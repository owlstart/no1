HRML 서식요소

이전 장에서는 HTML 스타일 속성 에 대해 배웠습니다 .
HTML은 또한 특별한 의미로 텍스트를 정의하기위한 특수 요소 를 정의합니다 .
HTML과 같은 요소 사용 <b>과 <i>같은 출력 서식을 굵게 또는 기울임 꼴 텍스트를.
서식 요소는 특수한 유형의 텍스트를 표시하도록 설계되었습니다.

  • <b> – 굵은 텍스트
    • <p><b>This text is bold.</b></p>

    <b>This text is bold.

  • <strong> –중요한 텍스트
    • <p><strong>This text is strong.</strong></p>

    This text is strong.

  • <i> –이탤릭체
    • <p><i>This text is italic.</i></<p>

    This text is italic.

  • <em> –강조된 텍스트
    • <p><em>This text is emphasized.</em></p>

    This text is emphasized.

  • <mark> –표시된 텍스트
    • <h2>HTML <mark>Marked</mark> Formatting</<h2>

    HTML Marked Formatting

  • <small> –작은 텍스트
    • <h2>HTML <small>Small<&slo;small> Formatting<&slo;<h2>

    HTML Small Formatting

  • <del> –삭제 된 텍스트
    • <p>My favorite color is <del>blue</del> red.</p>

    My favorite color is blue red.

  • <ins> –삽입 된 텍스트
    • <p>My favorite <ins>color</ins> is red.</p>

    My favorite color is red.

  • <sub> –첨자 텍스트
    • <p>This is <sub>subscripted</sub> text.</p>

    This is subscripted text.

  • <sup> –위첨자 텍스트
    • <p>This is <sup>superscripted</sup> text.</p>

    This is superscripted text.

    정의와 사용법
  • <address> 태그는 문서 또는 기사 작성자 / 소유자의 연락처 정보를 정의합니다.
  • <address> 요소가 <body> 요소 안에 있으면 문서의 연락처 정보를 나타냅니다.
  • 연락처 정보의 일부가 아닌 경우 우편 주소를 설명하는 데 사용해서는 안됩니다.
