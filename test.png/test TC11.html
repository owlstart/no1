CSS Layout - The position Property

position속성은
위치 결정 방법의 유형을 지정

위치 값

Value Description
static 위치 : 정적;기본적으로 정적 위치
relative 위치 : 상대;원래 위치를 기준으로 배치
fixed 위치 : 고정;
absolute 위치 : 절대;
sticky 위치 : 끈적임;

예

<style>
div.static{
position: static;
border:3px solid #00FA9A;
}

적용


<body>
<h2>제목</h2>
<div class="static">
내용 입력
</div>
