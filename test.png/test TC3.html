CSS box-sizing속성을 사용하면 요소의
전체 너비와 높이에 패딩과 테두리를 포함시킬 수 있습니다.

  • CSS 상자 크기 속성이없는 경우
  • CSS 상자 크기 속성

CSS 상자 크기 속성이없는 경우

기본적으로 요소의 너비와 높이는 다음과 같이 계산됩니다.
너비 + 패딩 + 테두리 = 요소의 실제 너비
높이 + 패딩 + 테두리 = 요소의 실제 높이
의미 :
요소의 너비 / 높이를 설정하면
요소의 경계와 패딩이 요소의 지정된
너비 / 높이에 추가되므로 요소가 설정 한 것보다 더 크게 나타납니다.
지정된 너비와 높이가 동일한
두 개의 <div> 요소를 보여줍니다.

.div1 {
width:  300px;
height:  100px;
border:  1px solid blue;
box-sizing:  border-box;
} .div2 {
width:  300px;
height:  100px;
padding:  50px;
border:  1px solid red;
box-sizing:  border-box;
}
두 div의 크기가 동일 합니다

같죠

스타일 boxSizing 재산권

boxSizing 속성 사용하면 특정 방식으로 영역에 맞도록 특정 요소를 정의 할수 있습니다.

2 개의 테두리가 있는 상자를 나란히 원하는경우
boxSizing을 "테두리 상자"로 설정 하면 됩지다.
브라우저가 지정된 너비와 높이로 상자를 렌더링 하고 상자인에 테두리와 패딩을 배치 합니다.

속성값

Value 값 Description 기술 JavaScript문법
content-box 기본값.CSS2.1에 의해 지정된 폭과 높이의 동작입니다.
지정된 너비와 높이(최소/최대 속성) 요소 내용상자의 너비와 높이에 각각 적용됩니다.
CSS구문
    box-sizing :
  • content-box :기본 너비/높이 속성 (최소/최대 속성)테두리와 패딩은 포합되지않음
  • border-box :너비/높이 속성(취소/최대 속성)테두리와 패딩 포함됨
  • initial:속성을 기본값으로 설정합니다
  • inherit:부모 요소에서 이 속성을 상속합니다
 JavaScript문법
   object.style.boxSizing = "border-box"
