배열

JavaScript 배열 객체

  • Array 객체는 단일 변수에 여러 값을 저장하는 데 사용
  • var cars = ["Saab", "Volvo", "BMW"];
  • 배열 인덱스는 0부터 시작
  • 배열의 첫 번째 요소는 0이고 두 번째 요소는 1입니다.

배열 속성

Property Description 예 JavaScript
constructor
  • Array 객체의 생성자 함수를 반환
  • 배열의 경우 생성자 속성은 함수 Array () {[native code]}를 반환
  • 숫자의 경우 생성자 특성은 함수 Number () {[native code]}를 리턴
  • 문자열의 경우 생성자 속성은 함수 String () {[native code]}을 반환
 배열의 생성자함수 반환 var cars = ["바나나", "포도", "사과"];
