JavaScript 는 모든 웹 개발자가 배워야 하는 3가지 언어중 하나입니다.

  • 웹 페이지의 내용을 정의 하는 HTML
  • 웹 페이지의 레이아웃을 지정하는 CSS
  • 웹 페이지의 동작을 프로그래밍 하는 JavaScript

JavaScript 가 사용되는 곳은 웹 페이지 뿐만 아닙니다.
많은 데스크탑 및 서버 프로그램이 JavaScript 를 사용합니다.
Node.js가 가장 잘 알려져 있습니다. MongoDB및 CouchDB와 같은 일부 데이터베이스도 프로그래밍 언어로 JavaScript를 사용 합니다.

JavaScript와 Java는 완전히 다른 언어 입니다.
JavaScript는 1995년 Brendan Eich의해 발명되었으며 1997년 ECMA-262는 표준 공식 이름 입니다.
ECMAScript는 언어의 공식 이름 입니다.

JS Versions장에서 다른 JavaScript버전에대한 자세한 내용을 읽을수 있습니다.

자바 스크립트 코드 블록

JavaScript 문은 중괄호 ({...}) 안에 코드 블록으로 함께 그룹화 할 수 있습니다.

코드 블록의 목적은 함께 실행될 명령문을 정의하는 것입니다.

블록으로 그룹화 된 명령문을 찾을 수있는 곳은 JavaScript 함수입니다.

예

<script> function myFunction() { document.getElementById("demo1").innerHTML =
"Hello Dolly!"; document.getElementById("demo2").innerHTML =
"How are you?"; } </script>

...하는 동안 조건문이 참인 동안 문장 블록을 실행하고 블록을 반복합니다.
