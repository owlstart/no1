HTML 양식 및 입력요소

Tag Description
<form> 사용자 입력을 위한 HTML 양식을 정의
<input> 입력 컨트롤 정의 input control

<form>

Attribute 속성 Value 값 Description 기술 설명
accept-charset character-set 문자 집합 양식 제출에 사용될 문자 인코딩을 지정
action URL 양식을 제출할 때 양식 데이터를 보낼 위치를 지정
autocomplete
  • on
  • 기본 사용자가 이전 입력한 값에다라 자동드로완성
  • off
  • 사용할때마다 각 필드에 값을 입력
 양식에서 자동완성 기능을 사용 할지 여부를 지정
enctype
  • application/x-www-form-urlencoded
  • multipart/form-data
  • text/plain
 양식 데이터를 서버에 제출할때 양식 데이터를 인코딩 하는 방법을 지정
method
  • get
  • post
 양식 데이터를 보낼때 사용할 HTTP 메소드를 지정
name text 양식 이름을 지정
novalidate novalidate 제출시 양식의 유효성을 검증하지 않도록 지정
target
  • _blank
  • _self
  • _parent
  • _top
 양식을 제출 한후 수신한 응답을 표시 할 위치를 지정
