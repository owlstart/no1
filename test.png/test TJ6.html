자바 스크립트 디스플레이 가능성

자바 스크립트는 다른 방식으로 데이터를 "표시"할 수 있습니다 :

  • , HTML 요소로 작성하여 innerHTML.
  • 을 사용하여 HTML 출력에 쓰기 document.write().
  • , 경고 상자에 쓰기 사용 window.alert().
  • 브라우저 콘솔에 쓰기 사용 console.log().

innerHTML 사용

HTML요소에 액세스하기 위해 JavaScript는 document.getElementById(id)메소드를 사용할 수 있습니다. 이 id속성은 HTML요소를 정의 합니다. 이 innerHTML속성은 HTML내용을 저의 합닏다.

예

<body>
<h2>My First Web Page</h2>
<p>My First Paragraph.</p>
<p id="demo"></p>
<script>
document.getElementById("demo").innerHTML = 5 + 6;
</script>
</body>
</html>

HTML 요소의 innerHTML 속성을 변경하는 것은 HTML로 데이터를 표시하는 일반적인 방법입니다.

document.write () 사용

테스트 목적으로 다음을 사용하는 것이 편리합니다 docunent.write().

예

<body>
<h2>My First Web Page</h2>
<p>My first paragraph.</p>
<p>Never call document.write after the document has finished loading.
It will overwrite the whole document.</p>
<script>
document.write(5 + 6);
</script>
</body>
</html>
