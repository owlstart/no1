제약조건

HTML5는 constraint validation 이라는 새로운 HTML 유효성 검사 개념을 도입

HTML 제약 조건 유효성 검사는 다음을 기반으로합니다.

  • 제약 조건 우효성 검사 HTML 입력 특성
  • 제약 조건 유효성 검사 CSS 의사 선택기
  • 제약 조건 유효성 검사 DOM 속성 및 메서드
Attribute Description
disabled 디세이벌드 disabled 입력 요소를 비활성화 하도록 지정
max max 입력 요소의 최대 값을 지정
min min 입력 요소의 최소값을 지정
pattern pattern 입력 요소의 값 패턴을 지정
requirde requerde 입력 필드에 요소가 필요함을 지정
type type 입력 요소의 유형을 지정

전체 목록을 보려면 HTML 입력 속성으로 이동하십시오 .
