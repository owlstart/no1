hosts 파일이란

host 파일은 컴퓨터의 호스트 이름(도메인 이름 : Domain Name 예를 들어, www.google.com)의 데이터베이스가 기록된 파일입니다.Windows의 경우 DNS보다 hosts가 우선시 되기 때문에 IP 주소와 해당 호스트 이름을 DNS에서가 아니라 hosts 파일에서 연결해 주는 역할을 하고 있습니다.

hosts 파일의 변조 여부를 확인하기

나중에 설명하지만, PC 사용자가 인지 못하는 사이에 이 hosts 파일의 내용이 몰래 위변조된 경우에 최악의 상황에 빠져 버릴 가능성이 있습니다. 사용자가 인터넷 뱅킹을 위해 거래 은행의 Web 사이트를 방문하여 ID와 비밀번호를 입력하여 로그인하면, 제3자에 의해서 자신의 계좌에서 돈이 빠져 나갈 수도 있습니다.

예를 들어, 이것은 악의를 가진 사람이 진짜 은행 Web 사이트를 꼭 빼닮은 가짜 은행 Web 사이트를 만들어 놓고, 사이트에 접속한 사용자의 PC를 감염시켜 hosts 파일의 내용을 바꾸어 변조해 두었다고 합시다.

그것을 눈치 못챈 사용자가 언제나처럼, 평소의 거래 은행 Web 사이트를 방문했다고 생각하지만, 실제로는 가짜 은행의 Web 사이트로 유도하도록 돼 있어서, 눈치 채지 못하고 로그인하면 피해를 받게 되는 것입니다. 이것은 인터넷 피싱(Phishing)에서 자주 사용되는 수법입니다.
